# Hola mundo

El más pequeño de los ejemplos de React se ve así:

```jsx
ReactDOM.render(
  <h1>Hello, world!</h1>,
  document.getElementById('root)
)
```

Este muestra un encabezado con el texto "Hello, world!" en la página.
